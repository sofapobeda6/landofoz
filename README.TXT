Бот телеграм, асинхронный, через Long polling с использованием Yandex GPT и Postgre.
Предназначен для анализа статистики видео.

Для того, что-бы работать с Yandex GPT и Telegram, соответствующие ключи должны быть в .env - файле.

==============================================
Установка:
git clone https://github.com/sofapobeda6/landofoz.git
cd ./landofoz
cp .env_example .env
Добавить ключи Yandex GPT и Telegram в .env

Запуск:
docker compose up -d

============================================
ИНСТРУКЦИЯ ПО ПОЛУЧЕНИЮ КЛЮЧА БОТА ТЕЛЕГРАМ

Найдите BotFather в Telegram (@BotFather)
Отправьте команду /newbot                
Введите имя бота                         
Введите username бота (должен заканчиваться на bot, например, mystats_bot)
Скопируйте полученный токен 
============================================

============================================
Инструкция по получению Yandex GPT API key

Перейдите в Yandex Cloud Console (console.cloud.yandex.ru) 
Создайте сервисный аккаунт:
Раздел "Сервисные аккаунты" - "Создать"
Укажите имя (например, gpt-service) 
Назначьте роль ai.languageModels.user

Создайте API-ключ:
Выберите созданный сервисный аккаунт
Вкладка "API-ключи" - "Создать ключ"

Скопируйте полученный API-ключ 
Скопируйте Folder ID (идентификатор каталога) из URL консоли или карточки каталога 
	Например для URL вида https://console.yandex.cloud/folders/b1gi545gfgckjah3bk/dashboard 
	b1gi545gfgckjah3bk и будет искомым Folder ID.

====================================================
Загрузка данных:
При первом запуске автоматически создаются таблицы и индексы из ./init.sql и,
	 если данных в таблицах нет, грузит данные в базу из videos.json.
Файл videos.json должен быть в ./data/videos.json
=====================================================

===========================================
Как работает:
Принмает через Long Polling сообщения от пользователя через бота телеграм.
Формирует на основе этого сообщения запрос к Yandex GPT (запрос состоит из системного промпта,
	 в котором задается контекст, находится в ./app/prompts.py и вопроса пользователя).
Получает в ответ от Yandex GPT текст sql-запроса к бд. 
Выполняет полученный запрос в БД и отправляет результат пользователю.
===========================================

==============================================
Что используется:
aiogram - для работы с Telegram bot API
asyncpg - для работы с postgreSql
aiohttp - для работы с Yandex GPT API
